### Summary

Use `<ZopaFooter />` to render Zopa's standard footer in a **Zopa application**.

### Examples

- Standard

```tsx
import { ZopaFooter } from '@zopauk/react-components';

<ZopaFooter />;
```

- Only with legal section

```tsx
import { ZopaFooter } from '@zopauk/react-components';

<ZopaFooter legalOnly />;
```
